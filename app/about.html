I am a Senior Java Developer/Architect with over 14 years of professional experience in Object Oriented Analysis and Design (OOAD), developing Java based systems which include multi-tiered and web based enterprise applications using J2EE technologies like Servlets, JSP, EJB(3.0,2.0), JDBC, JNDI, JMS, JPA, Java Beans, JAXB, JAXP(SAX/DOM), and JAX-WS.

I have experience developing end-to-end applications in both the public and private sector across multiple disciplines including: telecommunications, finance, marketing, academia, health, public health, state and federal government. Examples of applications developed include, but are not limited to: websites, web-based surveys, surveillance systems, training software, order entry systems, integration applications utilizing SOAP/REST, and ETL applications utilizing Rapid and Agile development methodologies.
